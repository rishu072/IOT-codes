# 🛠 Hardware Platform

## Arduino UNO

The Arduino UNO is one of the most popular microcontroller boards used for:

- Sensor Integration
- Home Automation
- Robotics
- IoT Prototyping
- Real-Time Monitoring Systems

### Specifications

| Feature | Details |
|----------|----------|
| Microcontroller | ATmega328P |
| Operating Voltage | 5V |
| Input Voltage | 7V - 12V |
| Digital I/O Pins | 14 |
| PWM Pins | 6 |
| Analog Inputs | 6 |
| Flash Memory | 32 KB |
| SRAM | 2 KB |
| Clock Speed | 16 MHz |

## 📏 Distance Measurement System

Uses an ultrasonic sensor to calculate object distance.

### Features
- Accurate distance detection
- Obstacle sensing
- Real-time monitoring

### Components
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Jumper Wires

## 🌧️ Rain Detection System

# 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/your-username/IOT-Codes.git
```

### 2. Open Arduino IDE

Open the desired project folder and load the `.ino` file.

### 3. Connect Arduino UNO

- Connect via USB
- Select Board: Arduino UNO
- Select COM Port

### 4. Upload Code

Click:

```text
Verify ✓
Upload →
```
